同学们：
通过两次以来大家的作业情况，我们对作业的形式做了一些改动：

1. 所有作业要求用C语言完成，提交 .c 格式的文件。

2. 下发作业时，每道题目在一个独立的文件夹里，文件夹里应有两个 c 文件：
    “[函数名].c”是大家需要完成的文件，请大家根据该文件中的提示，直接在该文件里需要填写的地方编写代码。不需要删除和改动任何已经存在的内容，不允许更改任何已经存在的函数、变量的标识符，也不允许更改文件名。仅允许使用已经加载的库的库函数。如果没有明确的提示，不允许自行编写额外的函数。完成作业后，直接将该文件打包提交。
    “test*.c”是测试文件，原则上不鼓励大家在完成作业前浏览，不允许对其进行更改。程序调试时，请将“test*.c”和“[函数名].c”置于同一路径下，并直接编译“test*.c”。

3. 上交作业时，请在elearning里上传一个文件名为学号的压缩包文件。压缩包只允许添加需要大家完成的“[函数名].c”以及report。
    例如，第二周作业：
    12307130073.zip
        - merge.c
        - removeNthFromEnd.c
        - report.txt / report.pdf

感谢大家配合
